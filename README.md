# Keyboard Helper
Learn automatic control mouse and keyboard via PyAutoGUI API.  
Using  tkinter package to make program interface.   

# Setup
* Anaconda 3  
* windows 10  
* `pip install pyautogui`  
* `pip install pyinstaller`  

# Build .exe program
```
pyinstaller -F -w custom_keyboard_event.py
```
